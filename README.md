# Django E‑Commerce Site

## Description
A full‑featured e‑commerce web app built with Django.

## Features
✔ Product listings  
✔ User registration/login  
✔ Shopping cart  
✔ Checkout & order tracking

## Tech Stack
Python, Django, HTML, CSS, JavaScript

## Setup
1. `git clone https://github.com/your‑username/django‑ecommerce‑site.git`
2. `cd django‑ecommerce‑site`
3. `pip install -r requirements.txt`
4. `python manage.py migrate`
5. `python manage.py runserver`
